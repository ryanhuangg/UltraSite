    Software and Documentation
    • Logging into Mathlab.
      1. Mac OS comes with the application "terminal". Simple click on it to run. Login to mathlab by typing ssh utorid@mathlab.utsc.utoronto.ca. Enter your password.
      2. Windows - you will need a SSH client. PuTTY is a good free one. You can download and install here: PuTTY. Use the same command as above to login to mathlab:
        ssh utorid@mathlab.utsc.utoronto.ca.
